# KitchenKart Splash Screen

## Purpose
Welcome users to KitchenKart while the app loads.

## Elements

- KitchenKart Logo (center)
- Tagline:
  "Smart Kitchen. Smarter Living."

- Background:
  White

- Accent Color:
  Green (#2E7D32)

- Loading animation:
  Small spinning circle

## Navigation

After 2 seconds:
→ Login Screen (if not logged in)
→ Dashboard (if already logged in)
